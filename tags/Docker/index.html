Docker 使用实例（部署 LNMP）

docker 参数说明

  • run 运行镜像，将创建容器
  • exec 运行容器（交互式）
  • start/stop/restart 启动/停止/重启容器
  • ps 查看运行中的容器
  • ps -a 查看全部容器
  • -d, –detach 在后台运行容器，并且打印容器id。
  • -i, –interactive 即使没有连接，也要保持标准输入保持打开状态，一般与 -t 连用。
  • -t, –tty 分配一个伪tty，一般与 -i 连用。

docker run 参数说明

更多参数说明见官网

  • -v 宿主机文件夹:容器文件夹 目录挂载，使用绝对路径。限制挂载文件夹在容器中只读，加 :ro
  • –name 容器名
  • -p 宿主机端口:容器端口 端口映射
  • –link
  • 最后一个参数为镜像名
